void tile ( Desktop *  d)

Tile desktop passed in as an argument (uses desktop's defined tiling layout).

Returns without any effect when:

  • current desktop is in the floating mode
{
dbg("tile(): IN\n");
if (d->tile_or_float == FLOAT)
return;
if (d->layout == MASTER) {
tile_layout_master(0);
} else if (d->layout == GRID) {
tile_layout_grid(0);
}
draw();
dbg("tile(): OUT\n");
}
